# Guía de Producto: Numisfera (App de Monedas Antiguas)

## 1. Visión General del Producto

Numisfera será una aplicación web de alta calidad dedicada a la exhibición, venta y futura subasta de monedas antiguas. La plataforma servirá como un punto de encuentro digital para una comunidad diversa, desde numismáticos profesionales hasta aficionados casuales, proporcionando herramientas robustas y contenido de alta fidelidad.

## 2. Audiencia Objetivo

La aplicación está diseñada para servir a tres grupos de usuarios principales:

*   **Coleccionistas Profesionales y Numismáticos:** Buscan una plataforma fiable con información detallada, certificaciones, historial de procedencia y herramientas de búsqueda potentes para gestionar y expandir sus colecciones.
*   **Inversionistas y Vendedores:** Requieren un mercado especializado y seguro para comprar y vender monedas como activos, con un enfoque en la transparencia y la autenticidad.
*   **Aficionados Casuales y Público General:** Se sienten atraídos por la historia y la estética de las monedas. Buscan una experiencia de navegación atractiva y educativa que les permita explorar el catálogo y aprender.

## 3. Alcance del MVP (Producto Mínimo Viable)

El lanzamiento inicial se centrará exclusivamente en ser la mejor plataforma de **Catálogo y Exhibición** del mercado. El objetivo es construir una base sólida y una comunidad comprometida antes de introducir funcionalidades transaccionales.

**Funcionalidades Clave del MVP:**
*   Visualizador de imágenes en alta resolución con zoom detallado.
*   Fichas de información completa por cada moneda: historia, material, ceca, año, grado de conservación, etc.
*   Sistema de búsqueda y filtrado avanzado (por época, región, material, etc.).
*   Posibilidad de que los usuarios creen "colecciones virtuales" o listas de deseos.

## 4. Arquitectura y Experiencia de Usuario

La aplicación se construirá sobre una base de **Sitio Web Tradicional (Multi-Page Application)**. Esto asegura una estructura robusta, URLs únicas para cada moneda (ideal para compartir y para el posicionamiento en buscadores - SEO) y una navegación clara.

Sobre esta base, se implementarán capacidades de **Aplicación Web Progresiva (PWA)** para el público móvil. Esto permitirá:
*   Una experiencia de usuario fluida y rápida en dispositivos móviles.
*   La posibilidad de "instalar" la app en la pantalla de inicio.
*   Capacidades sin conexión para consultar catálogos o colecciones previamente guardadas.

## 5. Principios de Diseño y Estética

El diseño visual será **Clásico y Elegante**. La interfaz de usuario buscará evocar la sensación de un museo, una biblioteca de investigación o una prestigiosa casa de subastas.

*   **Paleta de Colores:** Se explorarán tonos oscuros y sofisticados como verdes profundos, vinos, y maderas nobles, acentuados con colores neutros para garantizar la legibilidad.
*   **Tipografía:** Se dará preferencia a tipografías con serifa para los títulos y encabezados, transmitiendo una sensación de tradición y autoridad. La tipografía para el cuerpo del texto será limpia y muy legible.
*   **Foco:** El diseño siempre dará prioridad a las imágenes de las monedas, tratándolas como las piezas de arte que son.
